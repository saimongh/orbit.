Settings

  • Create: Press N or click the + button.
  • Drift Tracking: Set a "Catch Up Frequency" when editing a contact. Orbit will mark them as "Drifting" if you haven't logged a History item recently.
  • History: Use the History category to log interactions (In-person, Text, etc.). This resets the Drift counter.
  • Connections: Link contacts together using the Connections category.
  • Events: Use Events for birthdays (check "Reoccurring").
